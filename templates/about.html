About CardioPredict AI

An end-to-end machine learning pipeline for cardiovascular disease risk prediction — from raw clinical data to production-ready inference.

CardioPredict AI is a machine learning–powered web application designed to estimate the risk of cardiovascular disease using patient health metrics such as age, blood pressure, BMI, cholesterol, glucose levels, and lifestyle factors. The system was built using proper preprocessing, outlier removal, feature engineering, and cross-validated model selection.

Machine Learning Core

Random Forest and Gradient Boosting models trained on cleaned cardiovascular datasets with GridSearch hyperparameter tuning.

Clinical Health Metrics

Analyzes systolic/diastolic BP, BMI, cholesterol levels, glucose, smoking status, alcohol, and physical activity.

Preventive Screening Focus

Threshold adjusted to 0.4 to improve recall for high-risk patients — prioritizing early detection over raw accuracy.

Medical Disclaimer

This application is intended for educational and research purposes only. Predictions are generated by a machine learning model and should not be considered a substitute for professional medical advice, diagnosis, or treatment. Always consult a qualified healthcare provider.
